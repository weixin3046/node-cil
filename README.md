```
npm install --global chasingdream
```

# 创建项目
```
okii init [name] 来初始化项目
```
# 更新
 npm version <update_type>进行修改，update_type 有三个参数，
 patch：这个是补丁的意思，补丁最合适；
 minor：这个是小修小改；
 major：这个是大改咯；
 修改远端的版本npm publish;
